# Layout in CSS

Layout defines where the different elements will appear on the browser. 

## Display for controlling flow of elements

* `none`
* `block`
* `inline`
* `inline-block`
* `flex`
* `grid`
* `table`

## Layout with Position

* `static` (default)
* `relative`
* `absolute`
* `fixed`
* `sticky`
