# Display in CSS

The property display provides several values that allow you to control the flow of layout in interesting ways.

## List of values for `display`

/* precomposed values */
* `block`
* `inline`
* `inline-block`
* `flex`
* `inline-flex`
* `grid`
* `inline-grid`
* `flow-root`

/* box generation */
* `none`
* `contents`

/* multi-keyword syntax */
* `block flex`
* `block flow`
* `block flow-root`
* `block grid`
* `inline flex`
* `inline flow`
* `inline flow-root`
* `inline grid`

/* other values */
* `table`
* `table-row` /* all table elements have an equivalent CSS display value */
* `list-item`

/* Global values */
* `inherit`
* `initial`
* `revert`
* `revert-layer`
* `unset`
